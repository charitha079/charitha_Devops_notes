What is user management ??
User management in Linux is used to create, delete, and maintain user accounts.
It helps the system administrator to control who can access the system and what actions they can perform.

Basic Linux Commands for User Creation and Management:
-----------------------------------------------------

## useradd <user-name> ------> useradd charitha  {charitha user is created.. this is the command used to create a user}
## userdel <user-name> ------> userdel charitha  {charitha user is deleted}
## passwd  <user-name> ------> passwd charitha   {to create password for the user}
## adduser <user-name> ------> adduser charitha  {same as useradd for adding the user.
                                                  User-friendly version of useradd
                                                  > Automatically:
                                                  > Creates home directory
                                                  > Asks for user details (Full name, phone, etc.)
                                                  > Sets password during creation.}

>> we can check whether the user is created or not using "cat /etc/passwd" or "vim /etc/passwd".
>> similarly password info will be provided under the "cat /etc/shadow".

Enforcing password policies:
---------------------------
## change -M 90   -------------------> for changing the password.
## passwd -l <user-name>  -----------> This command is used for locking the user. As an administrative user you found someone is misusing their account
                                       then to lock that user you can use this command.
## passwd -u <user-name>  -----------> for unlocking the user.




GROUP IN USER MANAGEMENT:
-------------------------

A group is just a collection of users.

ðŸ‘‰ Instead of giving permissions to each user one by one, we:

GROUP MANAGEMENT COMMANDS:
--------------------------
## groupadd <group-name> ---------> groupadd linux { this command is to add a group}
## groupdel <group-name> ---------> groupdel linux { To delete the group}

>> Now what if you want to add a user in that linux group.. yes.. there is a command..
## usermod -aG <group-name> <user-name> ----------> usermod -aG linux charitha -----------> {under linux group charitha user is created}

>> where to checkout whether the group is created or not..

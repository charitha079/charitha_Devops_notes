BASIC LINUX COMMANDS:
====================
uname - for knowing the system informantion. (linux)
man   - (manual) it will give the informantion about the particular commands.
man uname - it'll give the information about uname.
whoami - to see which user logged in.
hostname - to know the server name.
hostname -I  - it shows the ip adresss
pwd - it'll show the present working directory.
su - to switch the user.
sudo - do as super user we can get good clarity about it in user management.
ls /- listing the files and directories.
cd /- to change the directory.
cd .. - cd .. moves you one directory back (up).
ls -l -long listing.
ls -lh - long listing in human readable format.
ls -lht -long listing..human readable..time order..
ls -lhtr -long listing.. human readable..but in reverse time order..
cat - cat is a command used to concatinating the file and printing the standard output..
cat /etc/os-release - for os details.
